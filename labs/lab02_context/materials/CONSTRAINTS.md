# Constraints (Must Follow)

아래 제약을 어기면 오답으로 간주한다.

1. **저장 포맷 고정**
   - `labs/lab02_context/target_app/data/tasks.json`의 JSON 스키마를 변경하지 않는다.
2. **CLI 호환성 유지**
   - 기존 명령(`list`, `add`, `done`)의 이름/인자 의미를 바꾸지 않는다.
3. **외부 의존성 금지**
   - 표준 라이브러리만 사용한다.
4. **대상 범위 제한**
   - 수정은 `labs/lab02_context/target_app/` 하위에서만 한다.
