# GitHub Copilot Coding Agent 실전 강의

이 저장소는  
**GitHub Copilot Coding Agent를  
자동완성 도구가 아닌 ‘AI 개발 워커’로 활용하는  
실전 강의 및 실습 자료**를 담고 있습니다.

---

## 이 저장소에서 다루는 것

- Copilot Coding Agent 실전 사용법
- 말로 개발하는 워크플로우
- 테스트 · 디버깅 자동화
- 기능 단위 작업 위임
- AI 워커 기반 개인/팀 개발 방식

---

## 대상 독자

- GitHub Copilot을 이미 사용해 본 개발자
- 혼자 개발하지만 팀 개발 방식이 필요한 사람
- AI로 개발 생산성을 구조적으로 높이고 싶은 실무자

---

## 저장소 구성

- `course/` : 강의 이론 및 실습 가이드
- `labs/` : 실습용 코드
- `prompts/` : 바로 쓰는 Copilot 프롬프트
- `templates/` : 팀/개인 워크플로우 템플릿
- `appendix/` : 부록 자료

---

## 사용 방법

1. `course/` 디렉터리의 강의 자료를 순서대로 읽습니다.
2. 각 강의에 대응하는 `labs/` 실습을 진행합니다.
3. `prompts/`와 `templates/`를 실제 프로젝트에 적용합니다.

---

## 핵심 철학

> Copilot을 쓰는 법이 아니라  
> **Copilot과 일하는 법**을 배웁니다.
